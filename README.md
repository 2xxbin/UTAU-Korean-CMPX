# UTAU 한국어 복합형 리스트 (Korean Complex, CMPX)

한국어를 다양한 형식으로 나누어서 사용하는 리스트입니다.

`- C`, `- V`, `- CV`, `CV`, `VV`, `VC`, `CC` 등의 형식을 지원합니다.

<br />

자세한 내용은 레포지토리의 위키 항목을 참고해주세요.
