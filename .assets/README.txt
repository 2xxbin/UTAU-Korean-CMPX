공식 홈페이지 :: https://github.com/2xxbin/UTAU-Korean-CMPX
위키 :: https://github.com/2xxbin/UTAU-Korean-CMPX/wiki

한국어 복합형 리스트에 관심을 가지고 다운로드 해주셔서 감사합니다!
복합형 음원 형식은 최소 단위의 음소를 기차처럼 연결해 사용하는 음원 형식입니다.

예시를 들어볼까요?
"안녕하세요" 라는 가사가 있습니다. 이걸 CMPX식으로 변환해볼까요?
"[- a / _a2 / _aN][N n / nY / _Yeo / _eo3 / _eoNG][nga][a s / se][e y / _Yo / o R]"
이렇게 변환되게 됩니다!

CMPX 리스트로 즐거운 우타우를 하실 수 있길 바라겠습니다!

100 ~ 160 BPM 사이의 8모라 BGM 사용을 추천드립니다.
너무 빠른 BGM을 사용하실 경우, 발음 추출이 원할하지 않을 수 있습니다.

리스트의 종류

- main :: 본 녹음 리스트입니다. 꼭 녹음을 해주세요. (94개/553에일리어스)
- option_consonant :: 영어 자음 등을 추가하는 선택 리스트입니다. (18개/110에일리어스)
- option_sepcial :: 특수 음소를 추가하는 선택 리스트입니다. (30개/84에일리어스)
- option_vowel :: 모음을 추가하는 선택 리스트입니다. (26개/140에일리어스)
